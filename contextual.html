Contextual Help

Contextual Help opens an Expert article in an overlay panel when the user clicks a help link. It’s perfect for “learn more” moments where you want to provide guidance without leaving the current screen.

Open Help Overlay

This button loads your Expert homepage in the overlay, but in a real deployment you’d point to a specific help article.
